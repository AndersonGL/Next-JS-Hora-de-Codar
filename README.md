# Next.js do Zero ao Avançado com Projetos 🚀

Curso prático focado no desenvolvimento de aplicações modernas com **Next.js**, abordando desde os conceitos fundamentais até implementações avançadas utilizadas no mercado.

O conteúdo é orientado a **projetos reais**, permitindo a aplicação prática do conhecimento em cenários próximos aos desafios profissionais.

---

## 📚 Sobre o Curso

Neste curso, o foco é aprender **Next.js do básico ao avançado**, construindo aplicações completas e bem estruturadas, utilizando ferramentas amplamente adotadas no mercado.

O aprendizado é progressivo, começando pelos fundamentos e avançando para arquiteturas mais robustas e profissionais.

---

## 🛠️ Tecnologias Utilizadas

- Next.js  
- React  
- Prisma ORM  
- NextAuth  
- Tailwind CSS  
- JavaScript / TypeScript  
- Node.js

---

## 🔑 Principais Conceitos Abordados

- Estrutura e fundamentos do Next.js  
- Sistema de rotas e layouts  
- Renderização SSR, SSG e ISR  
- Autenticação e autorização com NextAuth  
- Integração com banco de dados utilizando Prisma  
- Estilização moderna com Tailwind CSS  
- Boas práticas de organização e escalabilidade  
- Desenvolvimento de projetos completos  

---

## 🎯 Objetivo do Curso

Capacitar o desenvolvedor a criar aplicações **modernas, escaláveis e alinhadas às boas práticas do mercado**, utilizando o ecossistema do Next.js.

## 🚧 Status

- Instalação 

- node -v = verificar a versão do node

- npm -v =  Verificar a versão do npm

- npx create-next-app@latest = para instalar a versão mais recente

- "npx create-next-app (nome do projeto)"

- rfce para criar uma estrutura => instalando a extensão ES7+ React/Redux/React-Native snippets

- npm i  react-icons react-modal =>  Importa o react icons e o modal para dentro do projeto

- npx prisma migrate dev => Importa a estrutura do banco 

- npm run dev -> para iniciar o servidor
